# PawPal Connect Prototype

Interactive HTML prototype for a Pet Adoption and Donation social computing app.

## Demo pages

- `index.html` - interactive phone demo, suitable for GitHub Pages.
- `board.html` - overview board with all designed screens for report screenshots.

## Direct demo routes

The prototype supports direct links:

- `index.html#onboarding`
- `index.html#feed`
- `index.html#case`
- `index.html#tasks`
- `index.html#donation`
- `index.html#adoption`
- `index.html#notifications`
- `index.html#dashboard`
- `index.html#safety`
- `index.html#flow`

## Suggested presentation flow

Start helping today -> Feed -> Rescue case -> Help this case -> Claim task safely -> Updates -> Dashboard -> Safety center.

## GitHub Pages

Publish this folder as a static site:

1. Create a GitHub repository, for example `pawpal-connect-prototype`.
2. Upload all files in this folder.
3. Go to repository `Settings -> Pages`.
4. Choose `Deploy from a branch`, then select `main` and `/root`.
5. Open the Pages URL after deployment finishes.
